# Édito A1 法语学习网站

基于 Édito A1 教材整理的静态学习网站，无需服务器，直接用 GitHub Pages 托管。

## 页面

| 文件 | 说明 |
|------|------|
| `index.html` | 背单词系统（间隔复习 · 今日学习 · 阴阳性 · 拼写 · 听力） |
| `edito_a1_full.html` | 12个单元笔记合集（全文搜索 · 键盘切换） |
| `edito_a1_grammar_ref.html` | 语法手册（12章 · 冠词到速查总表） |

## 部署到 GitHub Pages

1. Fork 或新建仓库，上传这三个 HTML 文件
2. Settings → Pages → Source: `main` branch → 根目录 `/`
3. 保存后约1分钟，访问 `https://你的用户名.github.io/仓库名/`

## 数据说明

- 学习进度（SRS 间隔、今日词包）保存在**浏览器 localStorage**
- 数据绑定在当前浏览器，**换浏览器或设备需重新学习**
- 可在「自定义词」页面导出备份，手动迁移

## 功能

- **间隔复习**：SM-2 算法，❌🤔✅ 三档评分，自动调整复习间隔
- **今日学习**：每天选词 → 卡片认词 → 阴阳性 / 拼写 / 听力强化练习
- **100个动词**：含完整现在时变位、命令式、过去分词
- **559个内置词汇** + 自定义词汇支持批量导入
- **语音功能**：需要 Chrome 或 Edge（Safari 部分支持）
