# 🇫🇷 Édito A1 法语学习工具

基于 Édito A1 教材的静态学习网站，无需服务器，GitHub Pages 直接托管。

**[➜ 打开网站](https://你的用户名.github.io/edito-a1/)**

---

## 功能

### 📅 背单词（主页）
- 间隔复习算法（SM-2），❌🤔✅ 三档评分
- 今日学习：每天选词 → 卡片认词 → 阴阳性 / 拼写 / 听力练习
- 内置 559 个词汇 + 100 个动词（含完整变位）
- 支持自定义词汇和动词，批量导入
- GitHub Gist 云同步，跨设备保存进度

### 📖 全册笔记
- 12 个单元的词汇与句型整理
- 全文搜索，键盘左右切换单元

### 📝 语法手册
- 12 章 59 节，覆盖 A1 全部语法点
- 冠词 · 动词变位 · 时态 · 代词 · 句型 · 介词 · 时间天气 · 速查表

---

## 部署

```
1. 上传 index.html / edito_a1_full.html / edito_a1_grammar_ref.html 到仓库
2. Settings → Pages → Source: main branch
3. 访问 https://用户名.github.io/仓库名/
```

## 跨设备同步

在「自定义词」页面配置 GitHub Personal Access Token（仅需 `gist` 权限），数据自动同步到私有 Gist，换设备后拉取即可。Token 只存在本地浏览器，不上传到仓库。

---

语音功能需要 Chrome 或 Edge。
